Information

General Page

Here you can change the Icon of the Chatbot and its Name.


Corpus Page

Here you can create new Data which should be accessible through the Chat. The Data a Chatbot can provide thourgh the Chat consists of 3 Elements.

Intent - The reason why someone asks something
Utterance/s - Possible Questions to get an answer
Answer/s - The answer for possible questions

Add Intent -> Opens a Dialog where you can create a new Intent
Add Utterance -> Opens a Text Field where you can write a new Utterance
Add Answer -> Opens a Text Field where you can write a new Answer

It is also possible to edit an already created Intent.


Settings Page

Here you can setup which Role can acces which corpus Data
